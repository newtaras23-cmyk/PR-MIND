# PR-MIND — глосарій та house style (мовна консистентність)

Зафіксовані рішення після мовного аудиту (04.07.2026, `SPELLING-audit.md`). Мета — щоб майбутній контент писався консистентно без повторного узгодження цих же питань.

## Українська (правопис 2019, `mova.gov.ua`)

| Категорія | Обраний варіант | Приклад |
|---|---|---|
| Апостроф | типографський `’` (U+2019), НЕ прямий `'` | ім’я, з’являється, з’ясуйте |
| Тире | «—» (em dash) для пунктуаційних вставок, НЕ дефіс «-» | «...без відповіді — без управління» |
| Складні слова з іншомовним компонентом | з дефісом: `веб-`, `онлайн-`, `медіа-` | веб-розробка, онлайн-репутація, медіа-видимість |
| Минулий час жін. роду зворотних дієслів | форма на `-сь` | залишалась, лишалась, здавалась *(не «здавалася»)* |
| Бренд | завжди `PR-MIND` (великі літери, дефіс, без пробілів) | — |
| Кальки/русизми | заборонено: «на протязі», «приймати участь», «слідуючий» | використовувати: «протягом», «брати участь», «наступний» |

## Англійська (US spelling, AP Stylebook + Merriam-Webster)

| Категорія | Обраний варіант | Приклад |
|---|---|---|
| Дієслова -ize/-ise | `-ize` | organize, prioritize, optimize |
| -or/-our, -er/-re | американська форма | color, center *(якщо колись знадобляться — зараз не вжиті)* |
| Serial (Oxford) comma | **так, вживати завжди** у переліках із 3+ елементів *(відхід від чистого AP на користь фактичної консистентності сайту — див. SPELLING-audit.md)* | «privacy, clarity, and speed» |
| Апостроф | типографський `’` (U+2019), НЕ прямий `'` | seller’s, don’t |
| Тире | «—» (em dash) без пробілів навколо | — |
| Британізми | заборонено: programme, organisation, colour, licence, whilst, amongst, enquiry, realise, recognise | натомість: program, organization, color, license, while, among, inquiry, realize, recognize |
| website | одне слово, з малої літери в тексті | «website», НЕ «web site» |
